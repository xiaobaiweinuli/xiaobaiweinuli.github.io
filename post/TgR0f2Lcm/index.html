3、搜索引擎

最基础的找书方法，是在百度、必应、谷歌这样的搜索引擎里，用“书名+关键字”的方法搜索。

但是，你还可以用下面这些垂直搜索引擎，搜索更加精准。

| ▼ 搜索网站 | 简介 | | --- | --- | | 🌐 布克盘 | 布克盘聚合书籍数据库，用 zhelper search 搜索。 | | 🌐 知搜 | 收录了多家电子书网站，同时标记了网站的使用说明。 | | 🌐 搜书网 | 会显示搜索结果的类型，比如会显示网站是否能打开，是否需要收费等。 | | 🌐 淘链客 | 搜索内容来自搜索引擎、各大网盘或第三方网站，理工类书籍较多。 | | 🌐 鸠摩搜索 | 知名电子书搜索引擎，主要是收录各大网盘网站的内容结果。 | | 🌐 冀缘巧豫 | 搜索结果为阿里云网盘，支持上传电子书功能。 | | 🌐 UP云搜 | 搜索结果为阿里云网盘。 | | 🌐 TheFuture | 搜索结果为蓝奏云网盘。 | | 🌐 每日书单 | 搜索结果为城通网盘。 | | 🌐 莫若书单 | 搜索结果为城通网盘。 | | 🌐 书葵网 | 搜索结果以 BT 磁力。 | | 🌐 Taco搜索 | 除了支持搜索电子书，还支持“文档搜索”。 | | 🌐 FilePursuit | 神级资源搜索站，支持电子书、音乐、视频、软件、压缩包等，支持中文。 |
